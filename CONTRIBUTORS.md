## How to Be Featured

### Challenge Authors
Design and contribute a new challenge that gets merged into the repository. Your name will be added to the Challenge Authors section with a link to your GitHub profile and the challenge you created. See [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines.

When your challenge PR is merged, maintainers will:
- Add you to the Challenge Authors section
- Credit you in the challenge metadata
- Include your GitHub profile link

### Top Contributors
Make 10+ significant contributions including:
- Bug fixes and feature enhancements
- Validator improvements
- Documentation updates
- Test scenario additions
- Infrastructure improvements

### Community Leaders
Actively support the community through:
- Answering questions in Discord
- Reviewing pull requests
- Mentoring newcomers
- Organizing community events

## Recognition Criteria

**Challenge Author** — At least one merged challenge  
**Top Contributor** — 10+ merged PRs with significant impact  
**Community Leader** — Sustained community support (3+ months)  
**Maintainer** — Invitation-only based on sustained contributions and community trust

See [MAINTAINERS.md](MAINTAINERS.md) for governance details.
